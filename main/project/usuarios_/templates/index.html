Sobre o Projeto

Resumo

Com a popularidade das mídias sociais e marketplaces na última década, é cada vez mais comum que se encontre anúncios que contenham um produto que viole os direitos de propriedade intelectual de alguma marca, então é necessário que haja algum método que possa auxiliar na prevenção destes anúncios ilegais.

No entanto, o objetivo desta pesquisa é a implementação de um algoritmo de aprendizado supervisionado aplicado ao reconhecer padrões, a fim de identificar produtos falsificados anunciados em marketplaces.

Funcionalidades Sparrow

Extração de dados automatizada

O Sparrow faz uso de um famoso framework denominado Selenium, que é capaz de realizar extração de dados de uma maneira bem eficiente.

Sistema Especialista

O Sparrow é especialista no reconhecimento de anúncios que contenham produtos falsificados na internet. Suas decisões são fundamentadas através de uma grande base de conhecimento.

Apredizado Supervisionado

Para uma melhor acurácia nas classificações, o Sparrow conta com um sistema de aprendizado supervisionado. Ele aprende através do feedback do usuário.        

Cloud Computing

Todo o processamento de dados é realizado na núvem, dando ao Sparrow uma maior agilidade, escalabilidade e permitindo fácil acesso em qualquer dispositivo.  

Resultados Preliminares

+15.000 Anúncios coletados de um único produto.

O sistema de extração de dados conseguiu coletar em algumas horas, cerca de 15.200 anúncios suspeitos.

Base de conhecimento + 6.000 anúncios.

Para uma maior precisão, o Sparrow usou como base em suas decisões, cerca de + 6.000 anúncios, dentre eles anúncios originais e falsificados.

Na dúvida, o Sparrow pergunta para você.

Quando há duvidas na tomada de decisões, o Sparrow prefere ouvir o seu feedback.

Acurácia de + 90%

O Sparrow mostrou-se eficiente ao ter uma taxa de 93,6% de acerto nos produtos analisados. Dos 15.285 anúncios analisados, 69% eram originais e 31% falsos.

Projeto Sparrow mais a fundo

A venda de produtos falsificados na internet causa grande prejuízo financeiro às empresas. Segundo pesquisas divulgadas por alguns portais de notícias, estimativas indicam o Brasil é o 4º pais do mundo em que mais consome pirataria, tendo prejuízos anuais de aproximadamente R$ 115 bilhões de reais.

Empresas que são vítimas deste comércio ilegal, sofrem cada vez mais com a desvalorização da marca e produto. Com a popularidade das mídias sociais e marketplaces na última década, é cada vez mais comum que se encontre anúncios que contenham um produto que viole os direitos de propriedade intelectual de alguma marca, então é necessário que haja algum método que possa auxiliar na prevenção destes anúncios ilegais.

O objetivo desta pesquisa é desenvolver um sistema especialista que aplica Inteligência Artificial a fim de identificar anúncios e reconhecer produtos falsificados em marketplaces.

O propósito é auxiliar empresas na busca de produtos que violem os seus direitos de propriedade intelectual. Utilizando aprendizado de máquina supervisionado e técnicas de web scraping.
