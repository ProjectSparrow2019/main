Sobre o Projeto

Resumo

Este projeto de pesquisa consiste no desenvolvimento de uma ferramenta para reconhecimento de padrões de características de produtos anunciados em marketplaces, para fins de classificação de produtos originais e falsificados.

O objetivo desta pesquisa é a implementação de um algoritmo de aprendizado supervisionado aplicado ao reconhecer padrões, denominado Sparrow.

Projeto Sparrow mais a fundo

O Sparrow conta com sistema de autenticação, bastando apenas realizar o registro na página de cadastro. Após o envio dos dados, um e-mail de confirmação é enviado para o usuário. Na página inicial é possível ver dashboards informativos de todos os anúncios coletados em um determinado tempo, segregados entre produtos originais e falsificados O processo de busca de anúncios é acionado pelo usuário quando ele digita o nome do produto a ser coletado e processado, com isso o Sparrow busca por todos os anúncios relacionados àquele produto nos marketplaces. Quando um anúncio é coletado, o software seleciona o produto e o classifica se é original ou falso atribuindo-o uma porcentagem que revela o quanto o software acredita que aquele produto é falso, por exemplo: Produto A é falso com 83.32% de certeza, e caso seja detectado (por parte do usuário) um erro de classificação, pode-se submeter um feedback que é enviado diretamente para o treinamento da IA onde a informação será assimilada e utilizada nos próximos processamentos.
